# Bash Control Flow Cheat Sheet

```bash
if condition
then
  statements
[elif condition
  then statements...]
[else
  statements]
fi

for (( initialisation ; ending condition ; update ))
do
  statements...
done

case expression in
  pattern1 )
    statements ;;
  pattern2 )
    statements ;;
esac

while condition; do
  statements
done

```
